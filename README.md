# 电视影视语音控制源码实现

#### 介绍
电视端实现语音搜索影视、播放控制等，动态对接夏杰语音参考代码，可合入自己工程，即可实现语音控制功能。

#### 软件架构
软件架构说明


#### 使用说明

1.  下载编译工程，可合入到自己的影视工程。
2.  从当贝市场下载安装夏杰语音。
3.  运行VideoDemo，动态注册语音控制，即可实现语音功能。支持手机控制、微信小程序控制、智能音箱控制等等。
